# Виджет банковских операций
Виджет, который показывает банковские операций клиента, маскирует номера карт или счетов, позволять финансовым аналитикам быстро и удобно находить нужную информацию о транзакциях и проводить анализ данных.

## Требования
python = "^3.13"

flake8 = "^7.2.0"

black = "^25.1.0"

isort = "^6.0.1"

mypy = "^1.15.0"


## Установка 
В командной строке Вашего компьютера вставьте команду:
```sh
git clone git@github.com:Birdpersonal/bank-widget.git
```
Далее открываем PyCharm и нажимаем "Open"
[изображение](![image](https://github.com/user-attachments/assets/e0d70c99-7825-4a4c-be09-d372df099a67))
В открывшимся окне находим скаченный ранее файл "banc-widget"
